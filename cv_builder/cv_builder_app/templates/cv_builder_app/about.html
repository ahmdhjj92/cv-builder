About CV Builder

CV Builder is a a web application that allows users to enter their professional information and have them exported as a professional resume/CV.

It is an open source project dedicated to motivate and help people update their CV/resumes as it provides a better alternative to online CV makers.

Created by Ahmad El Hajj as an example project to learn web development, the app is written using 'Django' web framework and 'Bootstrap' CSS framework.
